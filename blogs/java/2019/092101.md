# 一、基本数据类型8种
## （1）4种整型

```Java
int             //4个字节
short           //2个字节
long            //8个字节
byte            //1个字节
```
## （2）2种浮点类型
```Java
float           //4个字节
double          //8个字节
```
## （3）char类型
```Java
char            //单个字符
```
# 二、运算符
## 移位运算符
```Java
>>              //将二进制位向右移
<<              //将二进制位向左移
```
